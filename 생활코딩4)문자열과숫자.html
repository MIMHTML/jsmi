데이터 타입(문자열과 숫자)

컴퓨터 프로그래밍에서는 데이터를 처리하는 것이

아주 아주 중요한 일이다.

데이터를 잘 처리하기 위해서,

데이터를 종류별로 잘 분류하기 위해서

상당히 많은 노력을 했고, 하고 있다.


JavaScript라는 컴퓨터 언어에는

어떤 형태의 데이터들이 있는가를 살펴보고

그 중에서 대표적인 것, 우리들이 익숙할만한 데이터인

'문자'와 '숫자'에 대해 깊게 이해하는 시간을 갖도록 하자.


*검색 - JavaScript data type

데이터 타입(자료형)의 종류

  1. Boolean
  2. Null
  3. Number(숫자)
  4. String(문자열)
  5. Symbol

데이터 타입의 종류는 이처럼 여러가지가 있다.

그 중에서 비교적 익숙한 데이터 타입인

문자열(String)과 숫자(Number)에 대해서 알아보자


*데이터 타입(숫자, Number)

JavaScript로 1+1을 출력하면 당연하게 2가 출력된다.

이 숫자(Number)라는 데이터 타입에서 아주 중요한 것은 연산이다.

그리고 이 더하기(+)는 '왼쪽에 있는 값과 오른쪽에 있는 값을 더해서 하나의 값을 만든다.'

라는 점에서 '이항 연산자'라고 부른다.

이항 연산자 중에서도 산술을 하는 것이기 때문에 '산술 연산자'라고 부른다.


*데이터 타입(문자)

문자는 따옴표(', ")로 시작해서 따옴표로 끝나면 된다.


문자를 처리할 때 사용할 수 있는 명령들 살펴보기

*검색 - JavaScript String

몇 가지 예시)
  1. string(문자열).length

    2. 문자열에 length를 붙여주면 문자의 글자수를 알려준다.

  2. string(문자열).toUpperCase()

    3. 소문자로 쓰여진 문자를 대문자로 출력한다.

  3. string(문자열).indexOf('찾고자 하는 값')

    4. 쓰여진 문자 내에서 내가 찾고자 하는 문자가 몇번째에 위치하는 지 알려준다.

    예를들어, 'hellow world'.indexOf('wolrd')라고 하면

    6을 출력하고 알파벳 world라는 문자가 쓰여져 있는 문자에서 6번째에 위치한다는 것을 출력값을 통해 알려준다.

  4. string(문자열).trim()

    5. 문자의 공백을 없애준다.
