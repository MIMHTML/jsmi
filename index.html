JavaScript

JavaScript란 HTML 위에서 동작하는 언어이다.

JS를 HTML과 퓨전시키는 방법은 굉장히 중요하다.

HTML에 JS를 넣는방법

웹브라우저한테 HTML의 코드로 '지금부터 JS가 시작됩니다.' 라고 알려줘야 된다.

그때 사용하는 태그

script 태그

이 script 태그는 태그 안쪽에 JS가 오도록 약속되어 있기 때문에 script 안쪽에 있는 코드를 JS로 해석한다.
