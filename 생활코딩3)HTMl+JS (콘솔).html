콘솔

지금까지 우리는 JS를 실행하기 위해서
웹페이지(파일)를 만들었다.
그런데 경우에 따라서는 꼭 파일이 아니더라도
그냥 어떤 코드를 실행해야 되는 아주 가벼운 상황들이 있을 수 있다.
그런 경우에는 console을 활용하면 된다.


이 말은 파일을 만들지 않고도 JS코드를 즉석에서 실행할 수 있다는 말이다.


우리가 계산이 필요할 때 계산기를 사용하듯이

데이터를 처리해야 되는 경우 콘솔을 사용하여 여러가지 데이터를 처리한다.

계산기 = 콘솔


일상에서 JS를 실행해야 되는 경우에
파일을 그때그때 만들어서 사용하는 것은 굉장히 귀찮은 일이다.
그 때 콘솔을 이용해 JS를 즉석으로 실행하면 편리하다.


간단한 예시) 문장에 들어있는 문자의 수를 셀 때

입력값 = alert('문장'.length)

출력값 = 경고창을 통해 문자의 수를 출력해준다.
